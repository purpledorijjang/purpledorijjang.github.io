App Introduction

앱 소개

Cafe Music Manager is an Android app that automatically plays YouTube Music according to user-defined schedules.

카페 음악 관리자는 사용자가 설정한 스케줄에 따라 자동으로 YouTube Music을 재생하는 안드로이드 앱입니다.

Key Features

주요 기능

  • Custom schedule setting
  • 사용자 정의 스케줄 설정
  • Automatic YouTube Music playback
  • 자동 YouTube Music 재생
  • Weather reflection feature
  • 날씨 반영 기능
  • Battery optimization exclusion support
  • 배터리 최적화 제외 지원
  • Accessibility service utilization
  • 접근성 서비스 활용
  • Detailed log recording
  • 상세한 로그 기록

Tech Stack

기술 스택

  • Language: Kotlin
  • 언어: Kotlin
  • Database: Room
  • 데이터베이스: Room
  • Asynchronous Processing: Coroutines
  • 비동기 처리: Coroutines

  • UI: RecyclerView, ConstraintLayout
